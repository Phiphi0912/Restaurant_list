# 餐廳清單
使用 Node.js 與 Express 所建立之餐廳清單，提供用戶搜尋、查閱相關餐廳資料

### 詳細頁面

## 功能
- 使用者能查看所有餐廳訊息
- 使用者能針對關鍵字進行搜尋
- 使用者能查閱餐廳詳細資訊

## 使用說明
1. 請先確認是否安裝 Node.js
2. 打開終端機並複製到本機電腦
```
https://github.com/Phiphi0912/Restaurant_list.git
```
3.進入專案資料夾
```
cd restaurant_list
```
4.安裝所需套件
```
npm install
```
5.開始運行
```
npm run start
```
6.若想針對專案進行開發人員測試
```
npm run dev
```
